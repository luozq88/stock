# 数据库配置说明

## 数据库文件
- **路径**: `data/stock_data.db`
- **类型**: SQLite3
- **模式**: WAL (Write-Ahead Logging)

## 数据表结构

### 1. stock_info (股票基本信息表)
| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | TEXT (PK) | 股票代码 (如: 000001.SZ) |
| symbol | TEXT | 股票代码 (纯数字) |
| name | TEXT | 股票名称 |
| area | TEXT | 所属地区 |
| industry | TEXT | 所属行业 |
| list_date | TEXT | 上市日期 |
| updated_at | TEXT | 更新时间 |

### 2. daily_data (日线行情数据表)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER (PK) | 自增主键 |
| ts_code | TEXT | 股票代码 |
| trade_date | TEXT | 交易日期 (YYYYMMDD) |
| open | REAL | 开盘价 |
| high | REAL | 最高价 |
| low | REAL | 最低价 |
| close | REAL | 收盘价 |
| pre_close | REAL | 昨收价 |
| change | REAL | 涨跌额 |
| pct_chg | REAL | 涨跌幅(%) |
| vol | REAL | 成交量(手) |
| amount | REAL | 成交额(千元) |

**索引**:
- `idx_daily_ts_code` - 按股票代码查询
- `idx_daily_trade_date` - 按交易日期查询
- `idx_daily_code_date` - 按股票代码+日期联合查询

### 3. tech_indicators (技术指标表)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER (PK) | 自增主键 |
| ts_code | TEXT | 股票代码 |
| trade_date | TEXT | 交易日期 (YYYYMMDD) |
| ma5 | REAL | 5日均线 |
| ma10 | REAL | 10日均线 |
| ma20 | REAL | 20日均线 |
| vol5 | REAL | 5日成交量均值 |
| calculated_at | TEXT | 计算时间 |

**索引**:
- `idx_tech_ts_code` - 按股票代码查询
- `idx_tech_trade_date` - 按交易日期查询
- `idx_tech_code_date` - 按股票代码+日期联合查询

## 数据来源
- **Tushare API**: 日线数据、股票基本信息
- **新浪财经**: 实时行情 (主数据源)
- **腾讯财经**: 实时行情 (备用数据源)

## 使用流程
1. **收盘后** (15:30): 运行 `main.py` 更新日线数据 + 计算技术指标
2. **午盘时** (11:30): 运行 `run_strategy.py` 获取实时行情 + 选股

## 选股策略条件
- 涨幅: 1.5% ~ 8%
- 现价 > 前一日 MA5
- 现价 > 前一日 MA10
- 现价 > 前一日 MA20
- 当日成交量(半天) > 前一日 VOL5 × 0.9
