# 股票数据获取程序规划

## 1. 技术栈

- **编程语言**: Python 3.8+
- **数据库**: SQLite
- **数据源**: 
  - Tushare (主要数据源)
  - 腾讯财经 (备用数据源)
  - 新浪财经 (备用数据源)

## 2. 核心功能

### 2.1 数据获取
- 从Tushare获取股票日线数据
- 获取最近60个交易日的K线数据
- 支持多数据源切换和容错

### 2.2 数据存储
- 使用SQLite数据库本地存储
- 设计合理的数据表结构
- 支持增量更新，避免重复数据

### 2.3 配置管理
- Tushare Token配置: `2f4f7b0dca606122c89b03503ebb70c4b26652328f446848a69e71e2`
- 数据库路径配置
- 交易日数量配置(默认60天)

## 3. 数据库设计

### 3.1 股票基本信息表 (stock_info)
```sql
CREATE TABLE stock_info (
    ts_code TEXT PRIMARY KEY,    -- Tushare股票代码
    symbol TEXT,                  -- 股票代码
    name TEXT,                    -- 股票名称
    area TEXT,                    -- 所在地域
    industry TEXT,                -- 所属行业
    list_date TEXT,               -- 上市日期
    updated_at TEXT               -- 更新时间
);
```

### 3.2 日线数据表 (daily_data)
```sql
CREATE TABLE daily_data (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    ts_code TEXT,                 -- 股票代码
    trade_date TEXT,              -- 交易日期
    open REAL,                    -- 开盘价
    high REAL,                    -- 最高价
    low REAL,                     -- 最低价
    close REAL,                   -- 收盘价
    pre_close REAL,               -- 昨收价
    change REAL,                  -- 涨跌额
    pct_chg REAL,                 -- 涨跌幅
    vol REAL,                     -- 成交量(手)
    amount REAL,                  -- 成交额(千元)
    UNIQUE(ts_code, trade_date)
);
```

### 3.3 索引设计
```sql
CREATE INDEX idx_daily_ts_code ON daily_data(ts_code);
CREATE INDEX idx_daily_trade_date ON daily_data(trade_date);
CREATE INDEX idx_daily_code_date ON daily_data(ts_code, trade_date);
```

## 5. 数据获取流程

```
1. 初始化数据库连接
2. 创建数据表(如果不存在)
3. 获取股票列表
4. 计算最近60个交易日的日期范围
5. 遍历股票列表:
   a. 检查本地是否已有数据
   b. 从Tushare获取日线数据
   c. 数据清洗和验证
   d. 保存到SQLite数据库
   e. 记录获取日志
6. 输出统计信息
```
