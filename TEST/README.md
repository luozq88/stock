# 策略2回测系统

## 策略概述

### 策略逻辑
1. **买入条件**：
   - 前一日涨幅：2% ≤ 涨幅 ≤ 8%
   - 前一日量比：成交量 ≥ VOL5 × 1.2
   - 当日涨幅：-2% ≤ 涨幅 ≤ 2%

2. **持仓周期**：2-4个交易日

3. **卖出逻辑**：
   - 在中枢相对高位卖出
   - 止损：-8%
   - 止盈：+15%
   - 强制在第4天卖出

## 注意事项

### 1. 数据要求
- 需要完整的日线数据
- 需要技术指标数据（MA5, MA10, MA20, VOL5）
- 数据库路径：`e:/other/stock/stock/data/stock_data.db`

### 2. 回测假设
- 可以按收盘价成交
- 不考虑滑点
- 佣金：万分之三
- 印花税：千分之一（卖出时收取）

### 3. 风险提示
1. **过度拟合风险**：回测表现好不代表实盘表现好
2. **市场变化风险**：策略可能不适应市场风格变化
3. **流动性风险**：小市值股票可能存在流动性问题
4. **黑天鹅风险**：极端市场情况可能导致策略失效
