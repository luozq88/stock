# 股票数据获取与选股策略系统

本项目使用Python获取股票数据并保存到本地SQLite数据库中，同时实现了多种选股策略。

## 功能特点

- 获取股票基本信息和日线数据
- 保存数据到本地SQLite数据库
- 实现多种选股策略：
  - 均线金叉策略
  - 成交量放大策略
  - 价格突破策略
- 选股结果统计和展示

## 环境要求

- Python 3.6+
- 依赖包：tushare, numpy, pandas

## 安装步骤

1. 克隆项目到本地
2. 安装依赖包：
   ```bash
   pip install -r requirements.txt
   ```
3. 在 `data_fetcher.py` 文件中替换为你自己的 tushare token：
   ```python
   ts.set_token('your_token_here')  # 需要替换为你自己的tushare token
   ```

## 使用方法

1. 运行主脚本：
   ```bash
   python main.py
   ```
2. 脚本会自动：
   - 初始化数据库，创建必要的表
   - 获取股票基本信息和最近30天的日线数据
   - 执行选股策略
   - 保存选股结果到数据库
   - 打印选股结果统计

## 数据库结构

- `stock_basic`：股票基本信息表
- `stock_daily`：股票日线数据表
- `strategy_result`：选股结果表

## 选股策略说明

1. **均线金叉策略**：当5日均线上穿10日均线时，认为是买入信号
2. **成交量放大策略**：当当日成交量是最近10天平均成交量的2倍以上时，认为是买入信号
3. **价格突破策略**：当当日收盘价突破最近20天的最高价时，认为是买入信号

## 注意事项

- 需要注册 tushare 账号并获取 token 才能使用数据接口
- 免费版 tushare 有调用次数限制，请合理使用
- 数据获取可能需要一些时间，请耐心等待
